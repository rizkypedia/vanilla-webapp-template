# Summary

Vanilla Web app using esbuild

## Install packages

````
npm i 
````

or 

````
npm install
````

## Start the app

````
npm run dev
````

Output looks like this and it also triggers a watcher

````
> esbuild-www@1.0.0 dev
> node esbuild.config.js

Serving at http://127.0.0.1:8000
````


Open browser and call http://localhost:8000
